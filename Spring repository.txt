*
Имя репозитория должно начинаться с имени Entity сущности NameReposytory (необязательно).
Второй Generic должен быть оберточным типом того типа которым есть ID нашей Entity сущности для которой создается репозиторий(обязательно).
Первый Generic должен быть объектом нашей сущности для которой мы создали Repository, это указывает на то, 
что Spring Data олжен предоставить реализацию методов для работы с этой сущностью (обязательно).
Мы должны унаследовать свой интерфейс от JpaRepository, иначе Spring Data непредоставит реализацию для нашего репозитория (обязательно).

**
Класс сервиса должен иметь аннотацию Service
